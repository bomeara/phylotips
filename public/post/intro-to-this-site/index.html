Intro to this site

2018-03-24 152 words 1 min read

Contents

Welcome to PhyloTips.info. From teaching my PhyloMeth course and working with students and colleagues in general, as well as the great example of Liam Revell’s Phytools Blog, I decided to make a site where I could post tips to deal with problems of people working on phylogenies. Since I mostly use R for this (though this may change as Julia, Python, etc. grow) I have created the site using the blogdown package of Yihui Xie | 谢益辉, which allows R Markdown to be beautifully rendered into a hugo website (I’m currently using the Even theme), with deployment on the free tier of netlify.

This makes overall deployment easy: blogdown:::new_post_addin() for a new post (if done within RStudio) and blogdown::build_site() to render the site, then git commit -m"updated site" -a and git push, and everything gets redeployed.
